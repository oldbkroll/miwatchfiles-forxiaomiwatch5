- [x] 全文件权限入口
- [x] 圆屏主页
- [x] 常用目录快捷入口
- [x] 只读目录列表
- [x] 表冠滚动
- [x] 设备诊断页
- [x] 低内存与 32 位 ABI 适配

- [x] 长按多选

- [x] 删除确认（M1C）

## M2：启动优化与简易文本查看/编辑

- [x] 采样图片预览
- [x] 图片缩放和平移
- [x] MIME 类型识别和“用其他应用打开”
- [ ] 启动速度与基础性能基线及优化
- [ ] 分段纯文本查看
- [ ] 简易文本编辑：覆盖当前文件、当前目录内另存为

M2 不开发内置音频播放器；音频和视频文件继续通过“用其他应用打开”调用手表已有应用。内置视频播放器移至 M4 可选评估。

验收：启动和目录加载有可重复的真机基线；文本查看不因大文件阻塞或 OOM；文本保存失败、取消或崩溃时原文件保持不变。

2026-07-19 已完成 M2 文本读取、事务写入、恢复和圆屏页面的本地实现与 JVM 聚焦测试；本次新设备会话无线 ADB 为 offline，尚未取得新的真机文本验收证据，因此以上三个 M2 验收项暂不勾选。详见 [`2026-07-19-m2-closeout.md`](../checkpoints/2026-07-19-m2-closeout.md)。
