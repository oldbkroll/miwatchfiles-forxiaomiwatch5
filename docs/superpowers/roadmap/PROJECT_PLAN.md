M3 的当前 Watch 5 证据已覆盖大任务 COPY/MOVE/DELETE 提醒、取消/返回、DELETE 二次确认链、普通小 COPY/MOVE/DELETE no-warning、冲突取消、替换全部和运行中 COPY 取消；前台服务与提示页状态详见

### M3 范围调整

- 不实现进程终止后的操作恢复、恢复日志、自动重试或任务持久化；`START_NOT_STICKY` 是明确的生命周期边界。
- 不把熄屏继续操作、Activity 重入后的任务恢复或后台驻留作为手表产品能力；不强制应用保持亮屏，也不改变系统调度策略。
- 不执行为凑阈值而构造的 `100 项`、`50 MiB` 或 `5000 项` 真机压力测试。大任务风险改由执行前提示覆盖：项目数达到 100 项或总容量达到 52,428,800 bytes（50 MiB）时，提示用户尽量保持亮屏并说明操作可能中断或失败。

目录继续使用固定的“文件夹优先、名称排序”。不新增收藏、最近目录、文件系统搜索、用户可选排序或独立文件属性页；文件详情页保留 MIME、大小、修改时间、读写状态和路径。
